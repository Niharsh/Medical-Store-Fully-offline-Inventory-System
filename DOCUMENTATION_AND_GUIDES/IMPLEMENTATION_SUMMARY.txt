PRODUCT RETURNS - HOW IT WORKS
──────────────────────────────

User Journey:
1. Create Invoice → Add items
2. For return items → Check "Mark as Return"
3. Select reason → Defective|Expired|Customer Request|Wrong Item|Damaged|Other
4. Submit → Invoice created with return flags
5. Invoice shows returns with [RETURN] badge and reason
6. Inventory automatically increases for returned items

Example Invoice:
  Item 1: Paracetamol - 10 tablets @ ₹50 = ₹500 (Sale)
  Item 2: Aspirin [RETURN] - 2 tablets @ ₹30 = ₹60 CR (Return)
          Reason: Defective
  
  Subtotal: ₹440 (500 - 60)
  Tax: ₹22
  TOTAL: ₹462

Inventory Impact:
  Paracetamol: -10 units (sold)
  Aspirin: +2 units (refunded)
