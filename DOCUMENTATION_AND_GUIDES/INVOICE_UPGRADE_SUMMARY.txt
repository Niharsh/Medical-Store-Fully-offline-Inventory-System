## How to Use

### Retail Invoice
```javascript

Bill To: John Doe
Phone: +91-9876543210

```

### Wholesale Invoice
```javascript
// Automatically shows:
Bill To (Buyer): ABC Medical Store
Address: 123 Main Street
Phone: +91-9876543210
GSTIN: (if available)
DL No: (if available)
```

**Detection Logic**: If invoice has `buyer_store_name`, `buyer_address`, `buyer_phone`, or `buyer_dl_number` → Wholesale mode

## Technical Details

### Calculations Fixed
```javascript
// OLD (Wrong)
const grandTotal = invoice.total_amount;  // ❌ ₹0.00

// NEW (Correct)
const grandTotal = taxableAmount + cgstAmount + sgstAmount;  // ✅ Correct
```

### Discount Calculation
```javascript
// Item-level discount (from items)
const itemDiscountAmount = sum(item.discount_amount);

// Invoice-level discount (percentage)
const invoiceLevelDiscountAmount = (subtotal * discount_percent) / 100;

// Total discount
const totalDiscountAmount = itemDiscountAmount + invoiceLevelDiscountAmount;
```

### Amount in Words Fix
```javascript
// OLD (Wrong)
const amountInWords = numberToWords(Math.floor(invoice.total_amount));  // ❌ "Zero"

// NEW (Correct)
const amountInWords = grandTotal > 0 
  ? numberToWords(Math.round(grandTotal))  // ✅ Correct
  : 'Zero';
```
